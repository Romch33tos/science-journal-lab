# Электронный научный журнал

Веб-приложение для управления статьями электронного научного журнала, разработанное с использованием Node.js, Express.js и MongoDB.

## Описание проекта

Приложение позволяет хранить, просматривать и искать научные статьи с возможностью фильтрации по названию и авторам. Каждая статья содержит информацию об авторах, дате публикации, содержании, тегах и пользовательских рецензиях с оценками.

### Основной функционал

- Отображение полного списка статей с краткой информацией (название, авторы, дата публикации)
- Поиск статей по названию
- Фильтрация статей по автору через выпадающий список

## Технологии

- **Node.js** - среда выполнения JavaScript
- **Express.js** - веб-фреймворк для создания серверной части
- **MongoDB** - нереляционная база данных для хранения документов
- **MongoDB Driver** - официальный драйвер для работы с MongoDB из Node.js

## Установка и запуск

### Предварительные требования

- Node.js версии 14 или выше
- MongoDB Community Server (запущенный на порту 27017)
- npm (входит в состав Node.js)

### Шаг 1: Клонирование репозитория

```bash
git clone https://github.com
cd science-journal-lab
```

### Шаг 2: Установка зависимостей

```bash
npm install
```

### Шаг 3: Заполнение базы данных

```bash
node seed.js
```
При успешном выполнении вы увидите сообщение:
```text
Подключение к MongoDB установлено.
Существующие документы удалены.
В коллекцию добавлено 5 документов.
Соединение с MongoDB закрыто.
```

### Шаг 4: Запуск сервера

```bash
node server.js
```

Сервер запустится на порту 3000:
```text
Сервер успешно подключился к MongoDB.
Сервер запущен на http://localhost:3000
```

### Шаг 5: Открытие приложения

Перейдите в браузере по адресу: [http://localhost:3000](http://localhost:3000)

## Использование приложения

### Просмотр списка статей
Нажмите кнопку **"Список статей"** для отображения всех публикаций. Таблица содержит:
- Порядковый номер
- Название статьи
- Список авторов (через запятую)
- Дату публикации в формате `ДД.ММ.ГГГГ`

### Поиск по названию
1. Введите ключевое слово или фразу в поле ввода.
2. Нажмите кнопку **"Поиск по названию"**.
3. Система выполнит регистронезависимый поиск и отобразит подходящие статьи.

### Поиск по автору
1. Выберите автора из выпадающего списка (список формируется автоматически).
2. Нажмите кнопку **"Поиск по автору"**.
3. Система отобразит все статьи, написанные выбранным автором.
