# Электронный научный журнал

Веб-приложение для управления статьями электронного научного журнала, разработанное с использованием Node.js, Express.js и MongoDB.

## Описание проекта

Приложение позволяет хранить, просматривать и искать научные статьи с возможностью фильтрации по названию, авторам, диапазону дат и рейтингу. Каждая статья содержит информацию об авторах, дате публикации, содержании, тегах и пользовательских рецензиях с оценками.

## Технологии

* **Node.js** — среда выполнения JavaScript
* **Express.js** — веб-фреймворк для создания серверной части
* **MongoDB** — нереляционная база данных для хранения документов
* **MongoDB Driver** — официальный драйвер для работы с MongoDB из Node.js

## Установка и запуск

Перед началом работы убедитесь, что в системе установлены:
* **Node.js** (версии 14 или выше)
* **npm** (пакетный менеджер, поставляется вместе с Node.js)
* **MongoDB Community Server** (должен быть запущен на стандартном порту `27017`)

### Шаг 1: Клонирование репозитория

Откройте терминал и выполните команды:
```bash
git clone https://github.com/Romch33tos/science-journal-lab
cd science-journal-lab
```

### Шаг 2: Установка зависимостей

Установите все необходимые пакеты проекта:
```bash
npm install
```

### Шаг 3: Заполнение базы данных

Запустите скрипт для генерации начальных данных в БД:
```bash
node seed.js
```

При успешном выполнении в консоли отобразится:
```text
Подключение к MongoDB установлено.
Существующие документы удалены.
В коллекцию добавлено 5 документов.
Соединение с MongoDB закрыто.
```

### Шаг 4: Запуск сервера

Запустите локальный сервер приложения:
```bash
node server.js
```

После этого в консоли отобразится лог успешного старта:
```text
Сервер успешно подключился к MongoDB.
Сервер запущен на http://localhost:3000
```

### Шаг 5: Открытие приложения

Перейдите в браузере по адресу: [http://localhost:3000](http://localhost:3000)

## Использование приложения

### Просмотр списка статей

Нажмите кнопку **«Список статей»**, чтобы отобразить все публикации. Таблица содержит следующие данные:
* Порядковый номер
* Название статьи
* Список авторов
* Дата публикации (в формате `ДД.ММ.ГГГГ`)
* Рейтинг и количество отзывов

### Поиск по названию

1. Введите ключевое слово или фразу в поле ввода.
2. Нажмите кнопку **«Поиск по названию»**.

Система выполнит регистронезависимый поиск и отобразит подходящие публикации.

### Поиск по автору

1. Выберите автора из выпадающего списка (формируется автоматически).
2. Нажмите кнопку **«Поиск по автору»**.

Система отобразит все статьи, написанные выбранным автором.

### Создание новой статьи

1. Нажмите кнопку **«Создать статью»** на главной странице.
2. Заполните поля формы:
   * Название
   * Авторы (через запятую)
   * Дата публикации
   * Содержание
   * Теги (опционально)
3. Нажмите кнопку **«Опубликовать статью»**.
